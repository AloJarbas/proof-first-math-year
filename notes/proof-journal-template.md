# Proof journal template

For the first week, the main win is not volume. It is learning a repeatable proof loop:

1. read the statement carefully,
2. rewrite hypotheses and conclusion,
3. draft a proof,
4. mark the gap honestly,
5. repair the proof into clean prose,
6. log the mistake pattern.

## Daily mini-log

At the end of each study day, add three lines:

```md
- What I proved or attempted:
- What still feels foggy:
- What I will retry from memory next:
```

## Week-1 checkpoint set

1. Statement anatomy: underline hypotheses, box the conclusion, and identify hidden quantifiers.
2. Definitions before proof: list the definitions that would have to appear in a rigorous proof.
3. One direct proof in full sentences.
4. One counterexample to a false universal claim.
5. One proof rewrite from a rough draft.
6. One 30-45 minute no-notes attempt.
7. One rewrite from memory the next day.

## Healthy week-1 output

- 2-3 finished proofs,
- 1 clean counterexample,
- 1 timed attempt,
- 1 rewrite from memory,
- 1 explicit error list.

## Anti-failure rules

- Do not count reading alone as progress.
- Do not leave proofs in half-symbolic shorthand.
- If stuck, rewrite the statement and definitions before hunting for tricks.
- If a proof fails, keep the failed attempt. It belongs in the journal.
